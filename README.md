# Movie Cast
Movie Cast - Movie & Tv Shows App. (Powered by [Tmdb API](https://www.themoviedb.org/documentation/api))

## Demo
[Click here.](http://movie-cast.herokuapp.com/)

## Features
- List Movies Page
- Tv Shows
- List Actors
- Search Dropdown

## Tech Stack
- Laravel
- Tailwind CSS
- Alpine.js
- Livewire
